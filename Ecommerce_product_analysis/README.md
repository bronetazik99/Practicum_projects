# E-commerce Product Analysis

## Goal
Analyze user behavior in the funnel view → cart → purchase and identify where users drop off.

## Dataset
E-commerce events dataset (views, cart, purchases).

## Main questions
- What does the conversion funnel look like?
- Where do users drop off?
- Which segments perform better or worse?
- What hypotheses can explain user behavior?

## Structure
- notebooks/analysis.ipynb — main analysis

## Tools
- Python (pandas, matplotlib)
- Jupyter Notebook
